# ☕ Coffee Shop App (React Native + Expo)

A simple mobile app built with **React Native** and **Expo** that displays a coffee shop menu and contact information.  
Perfect as a beginner project to learn the basics of React Native UI development.

---

## 📱 Features
- **Menu Screen** – Browse coffee items with names, prices, and descriptions.
- **Contact Us Screen** – View coffee shop address, phone number, and email.
- **Simple Navigation** – Switch between screens easily using React Navigation.
- **Responsive UI** – Works on both Android and iOS.

---

## 🛠 Tech Stack
- [React Native](https://reactnative.dev/) – Mobile app framework
- [Expo](https://expo.dev/) – Development and build tools
- [React Navigation](https://reactnavigation.org/) – Screen navigation
